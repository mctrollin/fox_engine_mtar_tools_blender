# Track Mapping File for MTAR Importer/Exporter
#
# This file defines bone renaming and transformation parameters for MTAR animations.
# Only bone mapping lines are processed; any @meta directives are ignored.
#
# Format:
#   source_track_name : target_bone_name ; param1=value1 ; param2=value2
#
# Supported transformation parameters are documented in the comments below.
# See README or source for details.
#
# ===== BONE MAPPING =====
#
# Each line maps a Fox Engine track name to a Blender bone and optionally
# specifies rotation offsets, axis remapping, space constraints, or IK.
# Comments beginning with '#' are ignored; blank lines are allowed.
#
# Examples:
#   SKL_000_DEF1_JNT : Root
#   SKL_001_DEF1_JNT : Spine1
#   SKL_002_DEF1_JNT : Spine2 ; offset_r=90,0,0,XYZ
#   SKL_003_DEF1_JNT : LeftShoulder ; map_r=y,-x,z
#   SKL_004_DEF1_JNT : Hand ; offset_r=90,0,0,ZXY ; offset_r=45,0,0,XYZ
#   Camera_Rot : camera_bone ; space_r=world
#   LookTarget_Rot : look_target ; as_ik_up=head,Z,2.0 ; space_ik=custom,torso ; offset_r=90,0,0,XYZ
#
# (Remnants of @meta directives have been removed; they no longer affect import/export.)
